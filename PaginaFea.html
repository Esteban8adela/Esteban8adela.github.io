What is HTML?

Is the standard language used to create and structure web pages.

Explain the head and the body tags.

- Head: Contains metadata and settings for the page

- Body: Contains the visible content of the page

Explain the section, article, aside, header, footer tag?

* Section: Groups related content together within a webpage.

* Article: A self-contained piece of content.

* Aside: A section of a webpage that is distinct from the main content but related to it.

* Header: A structural element that contains introductory content or navigational aids.

* Footer: A structural element that contains supplementary content or navigational aids.

Tag Selector

Targets all elements of a specific type

Class Selector

Targets elements with a specific class attribute.

Tag and Class Selector

Targets only a specific tag with a certain class.

Id Selector

Targets a single element with a unique id.

What is a class? What is an id? How are they related to CSS?

- Class: is an attribute that can be assigned to multiple elements.

- Id: is an attribute that can be assigned to only one element.

- Class and Id are related to CSS because they are used to target specific elements.
